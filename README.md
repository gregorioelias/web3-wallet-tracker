# Web3 Wallet Tracker

A simple Web3 portfolio tracker built with:

- Next.js
- Node.js
- Ethereum RPC
- Alchemy API

Features:

✔ Fetch wallet ETH balance  
✔ Fetch ERC20 tokens  
✔ Convert token balances  
✔ Basic portfolio UI  

Architecture:

Next.js frontend  
↓  
Node API  
↓  
Alchemy RPC  
↓  
Ethereum
